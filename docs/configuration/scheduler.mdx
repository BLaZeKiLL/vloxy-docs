# Scheduler

Scheduler settings are computed at runtime. these include
- **Meshing batch size** - batch size for chunk mesh generation job.
```csharp
Settings.Scheduler.MeshingBatchSize = 2 * Settings.Chunk.DrawDistance + 1;
```
- **Streaming batch size** - batch size for chunk data streaming job.
```csharp
Settings.Scheduler.StreamingBatchSize = 2 * Settings.Chunk.LoadDistance + 1;
```
