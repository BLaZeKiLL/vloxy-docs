# Overview

Vloxy Engine provides a **Scriptable Objects** which control the different aspects of the engine.

Noise settings is a seprate **Scriptable Objects** which is refrenced by the main settings scriptable object.

## Focus
**Focus Parameter** should be set as the transform around which the world should be stream on the world component

:::tip
You can leave this as null for fixed size worlds
:::

## Configuration

The configuration can be broken down in the following top level sections

- **[Noise Settings](./noise.mdx)**
- **[Chunk Settings](./chunk.mdx)**
- **[Renderer Settings](./renderer.mdx)**
- **[Scheduler Settings](./scheduler.mdx)**
