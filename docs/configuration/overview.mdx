# Overview

:::caution API Changes

Vloxy Engine is under development and API has not been finalized, docs may lag behind and API is subject to change until
the first stable release

:::

Vloxy Engine provides a hierarchy of **Scriptable Objects** which control the different aspects of the engine.

## Configuration

The configuration can be broken down in the following top level sections

- Noise Settings
- Chunk Settings
- Renderer Settings
- Scheduler Settings
